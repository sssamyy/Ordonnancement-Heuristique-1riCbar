# Ordonnancement-Heuristique-1riCbar
Implémentation en Python d’un algorithme heuristique pour résoudre le problème d’ordonnancement sur une machine unique avec temps de disponibilité et de traitement. L'objectif est de minimiser le temps moyen de complétion C̄ à l’aide d’une approche gloutonne basée sur les temps disponibles et de traitement.
